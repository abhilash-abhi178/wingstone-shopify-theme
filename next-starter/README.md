Wingstone Next.js Starter

A minimal Next.js 15 + Tailwind starter inspired by premium streetwear UI.

Quick start:

Install dependencies:

```bash
cd next-starter
npm install
```

Run dev server:

```bash
npm run dev
```

Notes:
- This starter includes `framer-motion`, `gsap`, and `lenis` for motion systems.
- Replace colors in `src/styles/globals.css` with your Coolors palette and add logos to `public/`.
- Components are minimal examples to iterate from.
